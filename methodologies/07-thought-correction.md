# 思想纠偏法

> 来源：《关于纠正党内的错误思想》（1929年12月）

## 核心原则

思想纠偏法是识别和纠正错误思想的方法论。核心框架：**识别错误表现 → 分析思想根源 → 提出纠正方法**

### 核心观点

1. **问题表现在行为，根源在思想**：只纠正行为不纠正思想，问题会反复
2. **系统分类**：不同类型的错误思想要用不同方法纠正
3. **对症下药**：找到思想根源，针对性解决
4. **教育与制度并重**：既要思想教育，也要制度约束

### 适用场景

- 团队出现思想问题，需要纠正
- 个人反思，寻找改进方向
- 组织文化建设
- 问题根源分析

## 分析框架

### 常见错误思想类型

| 类型 | 表现 | 根源 | 纠正方法 |
|------|------|------|----------|
| 单纯军事观点 | 只重业务，忽视政治/战略 | 缺乏全局观 | 提高站位，强化使命 |
| 极端民主化 | 无组织无纪律 | 缺乏规则意识 | 建立制度，严格执行 |
| 非组织观点 | 个人主义，不服从组织 | 个人利益至上 | 强化组织观念 |
| 绝对平均主义 | 搞平均，不承认差别 | 误解公平 | 认清公平与平均的区别 |
| 主观主义 | 脱离实际，主观臆断 | 缺乏调查研究 | 坚持实事求是 |
| 个人主义 | 只顾个人，不顾集体 | 私心重 | 强化集体意识 |
| 流寇思想 | 缺乏长远规划，打一枪换一个地方 | 缺乏战略定力 | 建立根据地思维 |
| 盲动主义 | 不顾条件，盲目行动 | 急于求成 | 认清客观规律 |

### 纠偏三步法

```
步骤一：识别错误表现
    │
    ├── 观察行为
    ├── 收集反馈
    └── 分析问题
    │
    ▼
步骤二：分析思想根源
    │
    ├── 为什么会这样？
    ├── 背后的思想是什么？
    └── 这种思想为什么产生？
    │
    ▼
步骤三：提出纠正方法
    │
    ├── 思想教育
    ├── 制度约束
    └── 实践锻炼
```

## 应用示例

### 示例 1：团队协作问题

**场景**：团队成员各自为政，缺乏协作。

**错误表现**：
- 信息不共享
- 推诿责任
- 各自优化局部，损害整体

**思想根源**：个人主义，只顾自己的一亩三分地

**纠正方法**：
- 思想教育：强调团队目标的重要性
- 制度约束：建立协作机制和考核标准
- 实践锻炼：安排跨部门项目，培养协作意识

### 示例 2：决策偏差

**场景**：项目决策脱离实际，屡屡失败。

**错误表现**：
- 决策拍脑袋
- 忽视基层反馈
- 盲目追求速度

**思想根源**：主观主义，脱离实际

**纠正方法**：
- 要求决策前必须有调研报告
- 建立基层反馈机制
- 设立决策复盘制度

## 详细指导

### 纠偏的原则

1. **治病救人**：纠偏的目的是帮助改进，不是打击
2. **实事求是**：对错误要有分析，不要无限上纲
3. **区别对待**：不同性质的问题用不同方法
4. **标本兼治**：既要纠正行为，也要纠正思想

### 思想分析的方法

| 方法 | 说明 |
|------|------|
| 五问法 | 连续问"为什么"，找到根源 |
| 换位思考 | 站在对方角度理解想法 |
| 历史分析 | 分析思想的产生过程 |
| 对比分析 | 与正确思想对比，找出偏差 |

### 纠偏的层次

```
行为层 → 纠正具体行为
    │
制度层 → 建立约束机制
    │
思想层 → 解决认识问题
    │
文化层 → 营造正确氛围
```

### 与其他方法论的配合

- **实事求是法**：分析错误思想是否符合实际
- **调查研究法**：了解真实情况，发现思想问题
- **组织纪律法**：用制度约束行为
