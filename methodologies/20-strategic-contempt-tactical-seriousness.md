# 战略藐视战术重视法

> 来源：《关于目前国际形势的几点估计》（1946年4月）、《和美国记者安娜·路易斯·斯特朗的谈话》（1946年8月）

## 核心原则

战略藐视战术重视法是毛泽东思想中处理**信心与谨慎**关系的辩证法。核心命题：**在战略上藐视敌人，在战术上重视敌人**

### 核心观点

1. **战略上藐视**：树立必胜信念，不被敌人吓倒
2. **战术上重视**：具体问题认真对待，不轻敌
3. **信心与谨慎的统一**：既有战略自信，又有战术谨慎
4. **一切反动派都是纸老虎**：从本质上看，敌人是可以战胜的

### 适用场景

- 面对强大对手，需要调整心态
- 制定战略规划，需要平衡信心与风险
- 执行具体任务，需要既坚定又谨慎
- 团队士气建设，需要鼓舞信心

## 分析框架

### 战略与战术的辩证统一

```
┌─────────────────────────────────────────────────────────────────┐
│                    战略层面                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    藐视敌人                                                     │
│    ├── 敌人本质上是虚弱的                                       │
│    ├── 我们一定能取得胜利                                       │
│    ├── 不被敌人的气势吓倒                                       │
│    └── 保持战略自信和定力                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    战术层面                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    重视敌人                                                     │
│    ├── 每个具体战斗都要认真准备                                 │
│    ├── 不打无准备之仗                                           │
│    ├── 不轻视任何一个对手                                       │
│    └── 具体问题具体分析                                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：战略判断

- 对手的本质是什么？
- 长期来看，谁是强者？
- 我们的优势在哪里？
- 为什么我们能赢？

### 步骤二：战术分析

- 当前对手有什么优势？
- 每个具体战斗如何打？
- 有哪些风险需要注意？
- 如何做好充分准备？

### 步骤三：统一协调

- 战略自信如何转化为战术行动？
- 战术谨慎如何不削弱战略信心？
- 如何在团队中传递正确的态度？

## 应用示例

### 示例 1：创业公司面对巨头

**场景**：创业公司与行业巨头竞争。

**战略藐视**：
- 巨头虽有资源，但可能官僚化、创新不足
- 我们更灵活、更专注
- 历史上很多小公司战胜了大公司

**战术重视**：
- 每个具体产品都要精心打磨
- 不与巨头正面硬刚
- 寻找巨头的薄弱环节

### 示例 2：面对困难任务

**场景**：团队接到一个艰巨的任务。

**战略藐视**：
- 这个任务虽然难，但一定能完成
- 相信团队的能力
- 保持必胜的信念

**战术重视**：
- 每个细节都要认真对待
- 做好充分的准备
- 预想各种可能的问题

## 详细指导

### "纸老虎"的辩证理解

| 层面 | 敌人是纸老虎 | 敌人不是纸老虎 |
|------|--------------|----------------|
| 战略层面 | ✅ 从本质看，可以战胜 | |
| 战术层面 | | ✅ 从具体看，要认真对待 |

### 两种错误倾向

| 错误倾向 | 表现 | 纠正方法 |
|----------|------|----------|
| 右倾机会主义 | 只看到敌人的强大，不敢斗争 | 战略上藐视，树立信心 |
| 左倾冒险主义 | 只看到敌人的弱点，轻敌冒进 | 战术上重视，认真准备 |

### 在团队中的应用

1. **鼓舞士气**：用战略藐视激励团队
2. **审慎执行**：用战术重视确保成功
3. **防止两个极端**：既不妄自菲薄，也不骄傲轻敌

## 经典语录

> 一切反动派都是纸老虎。看起来，反动派的样子是可怕的，但是实际上并没有什么了不起的力量。

> 从长远的观点看问题，真正强大的力量不是属于反动派，而是属于人民。

> 在战略上我们要藐视一切敌人，在战术上我们要重视一切敌人。

> 战争的胜负，主要地决定于作战双方的军事、政治、经济、自然诸条件，这是没有问题的。

> 然而不仅仅如此，还决定于作战双方主观指导的能力。

> 我们说不犯错误是不可能的，但错误犯得少一点，犯得小一点，这是可能的。
