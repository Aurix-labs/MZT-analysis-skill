# 矛盾分类处理法

> 来源：《关于正确处理人民内部矛盾的问题》（1957年2月27日）

## 核心原则

矛盾分类处理法的核心是：**不同性质的矛盾，用不同方法解决**

### 核心观点

1. **矛盾分类**：区分敌我矛盾和人民内部矛盾
2. **不同方法**：敌我矛盾用专政方法，人民内部矛盾用民主方法
3. **团结—批评—团结**：解决人民内部矛盾的公式
4. **防止混淆**：不能把人民内部矛盾当作敌我矛盾处理

### 适用场景

- 冲突处理，需要选择正确方法
- 团队管理，需要化解矛盾
- 利益协调，需要平衡各方
- 问题分类，需要区别对待

---

## 分析框架

### 两类矛盾的区别

| 特征 | 敌我矛盾 | 人民内部矛盾 |
|------|----------|--------------|
| 性质 | 对抗性矛盾 | 非对抗性矛盾 |
| 双方关系 | 敌对关系 | 根本利益一致 |
| 解决方法 | 专政方法 | 民主方法 |
| 处理原则 | 打击/防范 | 团结—批评—团结 |

### 解决人民内部矛盾的公式

```
团结 — 批评 — 团结
  │       │       │
  │       │       └─ 在新的基础上达到新的团结
  │       │
  │       └─ 通过批评和自我批评解决分歧
  │
  └─ 从团结的愿望出发
```

### 步骤一：判断矛盾性质

- 双方关系是什么？
- 根本利益是否一致？
- 矛盾是否具有对抗性？

### 步骤二：选择处理方法

| 矛盾类型 | 处理方法 |
|----------|----------|
| 敌我矛盾 | 坚决打击、有效防范 |
| 人民内部矛盾 | 民主讨论、说服教育 |
| 混合型矛盾 | 分清主次，分类处理 |

### 步骤三：执行并检验

- 处理后矛盾是否解决？
- 是否达到新的团结？
- 有无后遗症？

## 应用示例

### 示例 1：团队冲突处理

**场景**：团队成员因观点不同产生争执。

**矛盾性质判断**：
- 双方是否都希望团队好？是
- 根本利益是否一致？是
- 结论：人民内部矛盾

**处理方法**：
- 团结：肯定双方的出发点都是好的
- 批评：指出各自的问题（态度、方式等）
- 团结：达成共识，形成新的合作

### 示例 2：利益分配争议

**场景**：部门间因资源分配产生矛盾。

**错误做法**：
- 把争议方当作"敌人"对待
- 激化矛盾，造成对立

**正确做法**：
- 认识到这是人民内部矛盾
- 通过协商、讨论解决
- 寻求共赢方案

### 示例 3：竞争对手分析

**场景**：分析竞争对手的威胁。

**矛盾性质判断**：
- 与竞争对手的关系是对抗性的
- 属于"敌我矛盾"性质

**处理方法**：
- 认清竞争本质
- 制定竞争策略
- 不抱不切实际的幻想

## 详细指导

### 判断矛盾性质的标准

1. **根本利益是否一致**：这是最根本的判断标准
2. **矛盾是否具有对抗性**：能否通过协商解决
3. **矛盾双方的态度**：对方是否愿意合作
4. **发展趋势**：矛盾是否会激化或缓和

### 处理人民内部矛盾的原则

| 原则 | 说明 |
|------|------|
| 说服而非压服 | 通过讨论说服，不用权力压制 |
| 求同存异 | 在根本问题上一致，具体问题可以不同 |
| 批评与自我批评 | 双方都要反思，不是单方面批评 |
| 惩前毖后、治病救人 | 目的是改进，不是打击 |

### 防止两类矛盾混淆

- **左的错误**：把人民内部矛盾当作敌我矛盾，扩大打击面
- **右的错误**：把敌我矛盾当作人民内部矛盾，丧失警惕

### 与其他方法论的配合

- **矛盾分析法**：分析矛盾的性质和特点
- **阶级立场分析法**：分析各方立场，判断矛盾性质
- **群众路线法**：用民主方法解决人民内部矛盾

## 经典语录

> 敌我之间的矛盾是对抗性的矛盾。人民内部的矛盾，在劳动人民之间说来，是非对抗性的。

> 凡属于思想性质的问题，凡属于人民内部的争论问题，只能用民主的方法去解决，只能用讨论的方法、批评的方法、说服教育的方法去解决，而不能用强制的、压服的方法去解决。

> 在一九四二年，我们曾经把解决人民内部矛盾的这种民主的方法，具体化为一个公式，叫做"团结—批评—团结"。

> 解决人民内部矛盾，不能用咒骂，也不能用拳头，更不能用刀枪，只能用讨论的方法，说理的方法，批评和自我批评的方法。

> 不同质的矛盾，只有用不同质的方法才能解决。
