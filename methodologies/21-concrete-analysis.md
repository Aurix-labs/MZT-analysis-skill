# 具体问题具体分析法

> 来源：《矛盾论》（1937年8月）

## 核心原则

具体问题具体分析法是马克思主义的"活的灵魂"。核心命题：**不同质的矛盾，只有用不同质的方法才能解决**

### 核心观点

1. **矛盾的特殊性**：每个矛盾都有其特殊性，不能千篇一律对待
2. **反对公式化**：不能用同一个公式套用所有情况
3. **因地制宜**：根据具体情况制定具体方案
4. **活的灵魂**：这是所有方法论的前提

### 适用场景

- 作为所有分析方法的前提
- 面对新情况，需要创新方案
- 避免教条主义和经验主义
- 制定差异化策略

## 分析框架

### 具体问题具体分析的原则

```
┌─────────────────────────────────────────────────────────────────┐
│                    分析的三个层次                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    第一层：是什么？（具体表现）                                  │
│    ├── 这个问题的具体表现是什么？                               │
│    ├── 与其他问题的区别在哪里？                                 │
│    └── 有什么特殊性？                                           │
│                                                                 │
│    第二层：为什么？（根本原因）                                  │
│    ├── 为什么会出现这个问题？                                   │
│    ├── 根本原因是什么？                                         │
│    └── 这个原因与其他问题的原因有何不同？                       │
│                                                                 │
│    第三层：怎么办？（具体方案）                                  │
│    ├── 针对这个特殊情况，应该怎么办？                           │
│    ├── 别处的经验能否直接套用？                                 │
│    └── 需要做什么调整？                                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：识别特殊性

- 这个问题的特殊之处在哪里？
- 与过去遇到的问题有什么不同？
- 与别处的问题有什么不同？
- 有什么独特的背景和条件？

### 步骤二：分析原因

- 为什么会有这个特殊性？
- 根本原因是什么？
- 原因与特殊性之间有什么联系？

### 步骤三：制定方案

- 方案是否针对了特殊性？
- 是否简单套用了其他经验？
- 是否因地制宜、因时制宜？

## 应用示例

### 示例 1：问题诊断

**场景**：团队效率低下，需要找到原因。

**错误做法**：套用"标准方案"——加强培训、优化流程

**具体问题具体分析**：
- 这个团队效率低下的具体表现是什么？
- 是技能问题、流程问题、还是动力问题？
- 为什么会出现这个问题？（深入调查）
- 根据具体原因制定方案

### 示例 2：策略制定

**场景**：制定市场拓展策略。

**错误做法**：照搬其他市场的成功经验

**具体问题具体分析**：
- 这个市场有什么特殊性？
- 用户特点是什么？
- 竞争格局如何？
- 根据具体情况制定差异化策略

## 详细指导

### 与教条主义的区别

| 教条主义 | 具体问题具体分析 |
|----------|------------------|
| 照搬书本理论 | 理论与实际相结合 |
| 用同一个公式套用 | 因地制宜 |
| 忽视具体情况 | 分析特殊性 |
| 经验脱离实际 | 在实践中检验 |

### 与经验主义的区别

| 经验主义 | 具体问题具体分析 |
|----------|------------------|
| 照搬过去经验 | 分析新情况的特殊性 |
| 认为过去行的现在也行 | 认识到条件的变化 |
| 忽视条件差异 | 具体分析条件 |
| 经验变成教条 | 经验作为参考 |

### 分析的工具

1. **比较法**：与其他情况比较，找出特殊性
2. **历史法**：分析问题的发展过程
3. **背景法**：分析问题产生的背景条件
4. **调查法**：深入了解实际情况

## 经典语录

> 马克思主义的最本质的东西，马克思主义的活的灵魂，就在于具体地分析具体的情况。

> 不同质的矛盾，只有用不同质的方法才能解决。

> 用不同的方法去解决不同的矛盾，这是马克思列宁主义者必须严格地遵守的一个原则。

> 教条主义者不遵守这个原则，他们不了解诸种革命情况的区别，因而也不了解应当用不同的方法去解决不同的矛盾，而只是千篇一律地使用一种自以为不可改变的公式到处硬套。

> 研究问题，忌带主观性、片面性和表面性。
