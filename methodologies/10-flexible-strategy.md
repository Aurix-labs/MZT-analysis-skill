# 灵活战略法

> 来源：《抗日游击战争的战略问题》（1938年5月）

## 核心原则

灵活战略法的核心是：**在资源有限时，通过灵活机动实现战略目标**

### 核心观点

1. **保存自己，消灭敌人**：战争的基本原则
2. **集中与分散的统一**：该集中时集中，该分散时分散
3. **根据地建设**：建立稳固的基地，才能持续发展
4. **防御中的进攻**：在战略防御中实施战术进攻

### 适用场景

- 资源有限，需要灵活配置
- 面对强敌，需要以弱胜强
- 需要建立长期发展基础
- 环境多变，需要快速调整

## 分析框架

### 游击战争的基本原则

```
┌─────────────────────────────────────────────────────────────────┐
│                    保存自己，消灭敌人                             │
├─────────────────────────────────────────────────────────────────┤
│ • 保存自己是第一位的                                            │
│ • 在保存自己的前提下消灭敌人                                     │
│ • 积小胜为大胜                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 集中与分散的辩证关系

| 情况 | 策略 |
|------|------|
| 敌人分散 | 我集中兵力，各个击破 |
| 敌人集中 | 我分散行动，避免决战 |
| 发展机会 | 集中资源，重点突破 |
| 保存实力 | 分散风险，多点布局 |

### 根据地建设

- **为什么需要根据地**：没有根据地，就成了"流寇"
- **根据地的作用**：休整、补给、发展
- **如何建设根据地**：深耕一个领域/市场

### 步骤一：评估资源与约束

- 我有多少资源？
- 对手有多少资源？
- 有什么外部约束？

### 步骤二：选择战略姿态

- 什么时候进攻？
- 什么时候防御？
- 什么时候转移？

### 步骤三：建立根据地

- 在哪个领域/市场建立根据地？
- 如何深耕和巩固？

## 应用示例

### 示例 1：创业公司资源分配

**场景**：创业公司资金有限，如何分配？

**灵活策略**：
- 核心业务集中投入（集中）
- 探索业务小规模试验（分散）
- 确保现金流业务（根据地）
- 避免与巨头正面竞争（保存自己）

### 示例 2：项目管理

**场景**：项目资源紧张，多个需求并行。

**灵活策略**：
- 核心需求集中资源攻坚
- 非核心需求分散处理
- 建立基础能力（根据地）
- 不做无准备之仗

## 详细指导

### 灵活性的三个层次

1. **战略灵活性**：根据形势调整整体战略
2. **战术灵活性**：具体行动灵活机动
3. **应变能力**：对意外情况快速反应

### 建立根据地的原则

| 原则 | 说明 |
|------|------|
| 选择有利位置 | 扬长避短 |
| 深耕细作 | 不求大，求精 |
| 自我造血 | 能持续发展 |
| 可进可退 | 保持战略弹性 |

### 与其他方法论的配合

- **战略辩证法**：确定整体战略方向
- **聚焦突破法**：集中资源的方法
- **发展辩证法**：根据地从小到大的发展
