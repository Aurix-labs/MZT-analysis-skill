# MZT - Mao Zedong Thought 毛泽东思想辩证分析 Skill

[English](README_EN.md)

**星星之火，可以燎原。**

**MZT-analysis-skill——让 AI 掌握辩证思维的利器。**

MZT 将毛泽东选集中的 21 个核心方法论体系化为 AI Agent 可运用的思维能力。当 AI 遇到复杂问题时，它能够：

- **识别主要矛盾**，抓住问题的关键
- **实事求是**，基于实际情况分析
- **辨别立场**，理解利益相关方的诉求
- **制定策略**，独立自主、团结多数、聚焦突破
- **持续迭代**，在实践中检验和改进

**这不是简单的提示词模板，而是一套完整的辩证思维体系。**


## 项目定位

本项目专注于思维方法论的提炼与传播，不涉及任何政治立场。我们相信，优秀的思想智慧属于全人类，应当被传承和发扬。

**核心目标**：

1. **智慧传承**：将经过实践检验的思维方法体系化，让更多人能够学习和运用
2. **思维提升**：提供系统化的分析框架，帮助使用者培养深度思考和辩证分析能力
3. **开源共享**：作为开源项目，让优秀的方法论得以广泛传播，推动知识普惠

**使用原则**：
- 本工具提供的是**思维框架**，而非固定公式
- 使用者应保持**批判性思考**，结合实际情况灵活运用
- 方法论的价值在于实践，请遵循**具体问题具体分析**的原则

## 毛泽东思想经典语录

### 关于矛盾分析

> 在复杂的事物的发展过程中，有许多的矛盾存在，其中必有一种是主要的矛盾，由于它的存在和发展规定或影响着其他矛盾的存在和发展。

> 矛盾着的两方面中，必有一方面是主要的，他方面是次要的。

> 不同质的矛盾，只有用不同质的方法才能解决。

### 关于实事求是

> "实事"就是客观存在着的一切事物，"是"就是客观事物的内部联系，即规律性，"求"就是我们去研究。

> 没有调查，没有发言权。

> 一切结论产生于调查情况的末尾，而不是在它的先头。

### 关于实践与认识

> 通过实践而发现真理，又通过实践而证实真理和发展真理。

> 实践、认识、再实践、再认识，这种形式，循环往复以至无穷，而实践和认识之每一循环的内容，都比较地进到了高一级的程度。

### 关于发展战略

> 星星之火，可以燎原。

> 它是站在海岸遥望海中已经看得见桅杆尖头了的一只航船，它是立于高山之巅远看东方已见光芒四射喷薄欲出的一轮朝日，它是躁动于母腹中的快要成熟了的一个婴儿。

> 战争的目的不是别的，就是"保存自己，消灭敌人"。

### 关于群众路线

> 真正的铜墙铁壁是什么？是群众，是千百万真心实意地拥护革命的群众。

> 从群众中来，到群众中去。

> 一切群众的实际生活问题，都是我们应当注意的问题。

### 关于立场分析

> 谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。

> 革命党是群众的向导，在革命中未有革命党领错了路而革命不失败的。

### 关于工作方法

> 集中优势兵力、各个歼灭敌人的作战方法，不但必须应用于战役的部署方面，而且必须应用于战术的部署方面。

> 我们要下定决心，不怕牺牲，排除万难，去争取胜利。

### 关于宗旨信念

> 我们的共产党和共产党所领导的八路军、新四军，是革命的队伍。我们这个队伍完全是为着解放人民的，是彻底地为人民的利益工作的。

> 为人民利益而死，就比泰山还重；替法西斯卖力，替剥削人民和压迫人民的人去死，就比鸿毛还轻。

### 关于独立自主

> 我们希望有外援，但是我们不能依赖它，我们依靠自己的努力，依靠全体军民的创造力。

> 自力更生为主，争取外援为辅。

### 关于统一战线

> 所谓政治，就是把拥护我们的人搞得多多的，把反对我们的人搞得少少的。

> 在统一战线中，斗争是团结的手段，团结是斗争的目的。

## 使用方法

### 指令

| 指令 | 功能 |
|------|------|
| `/mzt` | 显式调用分析 — 进入对话式引导，输出结构化报告 |
| `/mzt-on` | 开启辩证思维模式 — Agent 在思考时自动融入辩证方法 |
| `/mzt-off` | 关闭辩证思维模式 — 恢复 Agent 默认思考方式 |

### 示例

**显式调用**：
```
用户: /mzt 我在考虑是否跳槽到一家创业公司

Agent: 让我来帮你分析这个职业决策问题...

[进入对话式引导，收集信息，输出结构化报告]
```

**开启辩证思维模式**：
```
用户: /mzt-on

Agent: 辩证思维模式已开启。我将在思考过程中运用毛泽东思想的方法论。
```

## 方法论体系

MZT 包含 **21 个方法论**，分为五个层级，形成完整的思维闭环。

### 第一层：哲学层

**世界观根基 — 所有分析的前提**

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| 实事求是法 | 从实际出发、反对主观主义 | 所有分析的前提 |
| 矛盾分析法 | 主要矛盾与次要矛盾 | 问题识别与优先级判断 |
| 具体问题具体分析法 | 不同矛盾用不同方法解决 | 避免教条主义 |

### 第二层：分析层

**问题诊断 — 识别关键因素**

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| 阶级立场分析法 | 利益决定立场、敌友辨别 | 利益相关方分析 |
| 调查研究法 | 没有调查没有发言权 | 信息收集、现状诊断 |
| 思想纠偏法 | 识别表现→分析根源→纠正 | 问题诊断、团队建设 |
| 矛盾分类处理法 | 敌我矛盾 vs 人民内部矛盾 | 冲突处理、问题分类 |

### 第三层：战略层

**方向规划 — 把握全局态势**

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| **独立自主法** | 把命运掌握在自己手中 | 战略支点建设、核心能力培养 |
| **统一战线法** | 团结多数、孤立少数 | 建立联盟、争取支持 |
| 战略辩证法 | 强弱转化、持久与速决 | 长期规划、竞争策略 |
| **战略藐视战术重视法** | 战略自信与战术谨慎 | 心态调整、风险控制 |
| 发展辩证法 | 小与大的转化、长远眼光 | 判断形势、评估潜力 |

### 第四层：执行层

**落地实施 — 具体操作迭代**

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| 聚焦突破法 | 集中资源、重点突破 | 资源分配、优先级执行 |
| 灵活战略法 | 保存自己消灭敌人 | 资源有限时的策略选择 |
| 统筹兼顾法 | 十大关系的平衡与协调 | 复杂系统设计、资源平衡 |
| **批评与自我批评法** | 团结—批评—团结 | 自我迭代、持续改进 |
| 认识循环法 | 感性→理性→实践检验 | 方案验证、迭代优化 |

### 第五层：保障层

**价值根基 — 根本宗旨与信念**

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| 群众路线法 | 从群众中来、到群众中去 | 需求分析、用户研究 |
| 宗旨导向法 | 根本宗旨、价值导向 | 使命愿景、意义建构 |
| 持之以恒法 | 坚定信念、持续努力 | 长期目标、困难任务 |
| 组织纪律法 | 原则性vs人情关系 | 团队协作、组织文化 |

> **注**：加粗为本版本新增的方法论

## 方法论间的辩证关系

方法论不是孤立的，它们之间存在辩证的对立统一关系：

| 方法论 A | 辩证关系 | 方法论 B | 说明 |
|----------|----------|----------|------|
| 聚焦突破 | ↔ | 统筹兼顾 | 集中与分散的辩证统一 |
| 独立自主 | ↔ | 统一战线 | 保持自我与团结他人的辩证统一 |
| 战略藐视 | ↔ | 战术重视 | 信心与谨慎的辩证统一 |
| 批评 | ↔ | 自我批评 | 对外与对内的辩证统一 |
| 实事求是 | ↔ | 矛盾分析 | 客观性与辩证性的统一 |

## 结构化报告章节

显式调用 (`/mzt`) 时，输出包含以下章节：

1. **问题定义** — 明确要解决的核心问题
2. **矛盾分析** — 主要矛盾、次要矛盾、矛盾的主要方面
3. **客观条件** — 当前环境和约束条件
4. **解决方案** — 基于分析提出的方案
5. **实践检验** — 如何验证和迭代方案
