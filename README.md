# MZT - 毛泽东思想辩证分析 Skill

基于毛泽东思想方法论的 AI Agent 辩证分析能力增强。

[English](README_EN.md)

## 安装

使用 [Skills CLI](https://github.com/vercel-labs/skills) 安装：

```bash
# 安装到当前项目
npx skills add your-github/mzt

# 全局安装到 Claude Code
npx skills add your-github/mzt -g -a claude-code

# 全局安装到 Cursor
npx skills add your-github/mzt -g -a cursor

# 安装到所有支持的 Agent
npx skills add your-github/mzt -g --all
```

## 使用方法

### 指令

| 指令 | 功能 |
|------|------|
| `/mzt` | 显式调用分析 — 进入对话式引导，输出结构化报告 |
| `/mzt-on` | 开启辩证思维模式 — Agent 在思考时自动融入辩证方法 |
| `/mzt-off` | 关闭辩证思维模式 — 恢复 Agent 默认思考方式 |

### 示例

**显式调用**：
```
用户: /mzt 我在考虑是否跳槽到一家创业公司

Agent: 让我来帮你分析这个职业决策问题...

[进入对话式引导，收集信息，输出结构化报告]
```

**开启辩证思维模式**：
```
用户: /mzt-on

Agent: 辩证思维模式已开启。我将在思考过程中运用毛泽东思想的方法论。
```

## 方法论体系

MZT 包含 16 个方法论，分为三层架构：

### 核心哲学层

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| 矛盾分析法 | 主要矛盾与次要矛盾 | 复杂问题拆解、优先级判断 |
| 实事求是法 | 从实际出发、反对主观主义 | 现状诊断、方案评估 |
| 认识循环法 | 感性→理性→实践检验 | 方案验证、迭代优化 |
| 发展辩证法 | 小与大的转化、长远眼光 | 判断形势、评估潜力 |

### 分析诊断层

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| 阶级立场分析法 | 利益决定立场、敌友辨别 | 利益相关方分析、谈判博弈 |
| 调查研究法 | 没有调查没有发言权 | 信息收集、现状诊断 |
| 思想纠偏法 | 识别表现→分析根源→纠正 | 问题诊断、团队建设 |
| 矛盾分类处理法 | 敌我矛盾 vs 人民内部矛盾 | 冲突处理、问题分类 |

### 策略执行层

| 方法论 | 核心要点 | 适用场景 |
|--------|----------|----------|
| 战略辩证法 | 强弱转化、持久与速决 | 长期规划、竞争策略 |
| 灵活战略法 | 保存自己消灭敌人 | 资源有限时的策略选择 |
| 聚焦突破法 | 集中资源、重点突破 | 资源分配、优先级执行 |
| 统筹兼顾法 | 十大关系的平衡与协调 | 复杂系统设计、资源平衡 |
| 群众路线法 | 从群众中来、到群众中去 | 需求分析、用户研究 |
| 组织纪律法 | 原则性vs人情关系 | 团队协作、组织文化 |
| 持之以恒法 | 坚定信念、持续努力 | 长期目标、困难任务 |
| 宗旨导向法 | 根本宗旨、价值导向 | 使命愿景、意义建构 |

## 结构化报告章节

显式调用 (`/mzt`) 时，输出包含以下章节：

1. **问题定义** — 明确要解决的核心问题
2. **矛盾分析** — 主要矛盾、次要矛盾、矛盾的主要方面
3. **客观条件** — 当前环境和约束条件
4. **解决方案** — 基于分析提出的方案
5. **实践检验** — 如何验证和迭代方案
