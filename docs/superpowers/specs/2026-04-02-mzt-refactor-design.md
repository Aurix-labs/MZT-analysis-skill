# MZT Skills 全面重构设计文档

## 概述

**项目名称**：MZT Skills 全面重构

**目标**：基于毛泽东选集原文评估结果，对 MZT 方法论体系进行全面重构，包括架构调整、新增方法论、来源修正。

**核心理念**：将毛泽东思想方法论体系从 5 层架构升级为 6 层架构，新增"认识层"，补充"主观能动性法"和"领导方法法"两个重要方法论，修正来源标注，使体系更加完整、科学。

## 二、新增方法论

### 2.1 主观能动性法

**来源**：《论持久战》（1938年5月）

**核心命题**：在客观条件许可的范围内，充分发挥主观努力，争取最好结果

**核心观点**：
1. 自觉的能动性：人类的特点是自觉的能动性，这是区别于动物的本质
2. 三元结构：客观事实 → 正确思想 → 正确行动
3. 舞台与导演：在既定舞台上，可以导演出有声有色的活剧
4. 优劣转化：正确的主观指导可以化劣势为优势，化被动为主动

**适用场景**：
- 资源有限时如何争取最好结果
- 评估目标是否可行
- 区分"不可能"与"困难但可能"
- 战略规划与决策制定

**哲学意义**：
- 主观能动性是毛泽东对马克思主义认识论的重要发展
- 它解决了"客观条件"与"主观努力"的辩证关系
- 是连接"实事求是"与"实践论"的关键桥梁

### 2.2 领导方法法

**来源**：《关于领导方法的若干问题》（1943年6月）

**核心命题**：一般与个别相结合，领导与群众相结合

**核心观点**：
1. 两个结合：一般号召与个别指导相结合，领导骨干与广大群众相结合
2. 从群众中来，到群众中去：集中群众意见，形成决策，再回到群众中坚持下去
3. 突破一点，推动全局：取得经验后推广，实现全面突破
4. 一个中心工作：在一定时间内只能有一个中心工作，统筹全局

**适用场景**：
- 团队管理与项目推进
- 变革管理与落地执行
- 优先级管理与资源分配
- 经验总结与推广复制

**与群众路线法的边界**：
- 群众路线法侧重"价值立场"（为什么人）
- 领导方法法侧重"工作方法"（怎么做）

## 三、层级调整

### 3.1 调查研究法：分析层 → 哲学层

**调整理由**：
- 《反对本本主义》开篇即言"没有调查，没有发言权"，这是认识论命题
- 调查研究触及"知"与"行"的关系，属于认识论范畴
- 毛泽东明确指出调查是"解决问题"的前提，是认识世界的根本途径

### 3.2 批评与自我批评法：执行层 → 认识层

**调整理由**：
- 批评与自我批评的核心是"团结—批评—团结"的循环，本质是认知迭代
- 《关于纠正党内的错误思想》指出其目的是"使党员干部的思想正确"
- 这是"自我革命"机制，属于认知校正范畴，而非单纯的执行工具

### 3.3 持之以恒法：保障层 → 执行层

**调整理由**：
- 《愚公移山》核心是"下定决心，不怕牺牲，排除万难，去争取胜利"
- 这是一种执行品质，是落地实施过程中需要的坚持精神
- 功能定位更偏向执行过程，而非价值根基

## 四、来源标注修正

### 4.1 群众路线法

**当前标注**：《关心群众生活，注意工作方法》（1934年1月27日）

**修正为**：《关心群众生活，注意工作方法》（1934年1月27日）、《关于领导方法的若干问题》（1943年6月）

**修正理由**：
- "从群众中来，到群众中去"的完整表述出自《关于领导方法的若干问题》(1943)
- 该文献是群众路线方法论的核心来源，不应遗漏

### 4.2 独立自主法

**当前标注**：《井冈山的斗争》（1928年11月）、《论持久战》（1938年5月）

**修正为**：《井冈山的斗争》（1928年11月）、《统一战线中的独立自主问题》（1938年11月）、《论持久战》（1938年5月）

**修正理由**：
- 《统一战线中的独立自主问题》是毛泽东专门论述独立自主问题的核心文献
- 标题直接点明主题，核心论述"保存统一战线中的独立自主"出自此文
