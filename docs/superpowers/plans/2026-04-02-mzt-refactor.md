```markdown
# 主观能动性法

> 来源：《论持久战》（1938年5月）

## 核心原则

主观能动性法是毛泽东对马克思主义认识论的重要发展。核心命题：**在客观条件许可的范围内，充分发挥主观努力，争取最好结果**

### 核心观点

1. **自觉的能动性**：人类的特点是自觉的能动性，这是区别于动物的本质
2. **三元结构**：客观事实 → 正确思想 → 正确行动
3. **舞台与导演**：在既定舞台上，可以导演出有声有色的活剧
4. **优劣转化**：正确的主观指导可以化劣势为优势，化被动为主动

### 适用场景

- 资源有限时如何争取最好结果
- 评估目标是否可行
- 区分"不可能"与"困难但可能"
- 战略规划与决策制定

---

## 分析框架

### 客观条件与主观努力的关系

```
┌─────────────────────────────────────────────────────────────────┐
│                    客观条件 → 主观努力 → 结果                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    客观条件（舞台）                                              │
│    ├── 提供可能性的边界                                          │
│    ├── 不能超越的限度                                            │
│    └── 可以被认识和利用                                          │
│           │                                                     │
│           ▼                                                     │
│    主观努力（导演）                                              │
│    ├── 从客观事实引出思想                                        │
│    ├── 形成正确的计划、方针、策略                                 │
│    └── 付诸行动，争取最好结果                                     │
│           │                                                     │
│           ▼                                                     │
│    结果（戏剧）                                                  │
│    ├── 在限度内争取最好                                          │
│    ├── 可能化劣势为优势                                          │
│    └── 坐着不动只有被灭亡                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：认识客观条件

- 当前有哪些客观条件？
- 这些条件提供了什么可能？
- 有哪些不可逾越的限度？

**关键问题清单**：
- [ ] 我是否准确认识了客观事实？
- [ ] 我是否混淆了"困难"与"不可能"？
- [ ] 我是否发现了条件中的有利因素？

### 步骤二：形成正确思想

- 从客观事实出发，引出思想、道理、意见
- 提出计划、方针、政策、战略、战术
- 确保思想符合客观实际

### 步骤三：付诸行动

- 将思想转化为行动
- 在行动中检验思想
- 根据反馈调整策略

### 步骤四：争取最好结果

- 不做超越条件的空想
- 在限度内争取最好
- 通过主观努力改变优劣对比

---

## 应用示例

### 示例 1：创业决策

**场景**：创业公司资源有限，面临强大竞争对手。

**客观条件分析**：
- 资源不如竞争对手（限度）
- 但市场足够大，细分机会存在（可能）
- 团队有独特优势（有利因素）

**主观努力方向**：
- 避开正面竞争，选择细分市场
- 发挥团队优势，快速迭代
- 在局部形成优势，逐步扩大

**结果**：虽整体劣势，但在细分市场取得领先。

### 示例 2：个人成长

**场景**：想要转行，但缺乏相关经验。

**错误认识**：认为"不可能"，放弃努力

**正确认识**：
- 客观条件：缺乏经验是事实，但不是不可改变
- 主观努力：可以学习、实践、积累
- 结果：通过持续努力，成功转行

---

## 详细指导

### 应用技巧

1. **区分可能与不可能**：客观条件不允许的是不可能，只是困难的是可能
2. **从事实出发**：思想必须建立在客观事实基础上
3. **主动作为**：坐着不动只有被灭亡，必须主动争取
4. **动态调整**：主观指导要随着客观条件变化而调整

### 常见误区

| 误区 | 表现 | 纠正方法 |
|------|------|----------|
| 乱撞乱碰 | 脱离客观条件，凭空想象 | 从客观事实出发，形成正确思想 |
| 坐着不动 | 有条件不利用，消极等待 | 主动作为，否则只有被灭亡 |
| 只谈条件 | 强调客观限制，放弃主观努力 | 在限度内争取最好结果 |
| 超越限度 | 不顾客观条件，盲目蛮干 | 认清边界，量力而行 |

### 与其他方法论的配合

- **实事求是法**：主观能动性必须建立在对客观事实的正确认识上
- **认识循环法**：主观认识要在实践中检验和深化
- **战略辩证法**：通过主观努力实现强弱转化

---

## 经典语录

> 这种能动性，我们名之曰"自觉的能动性"，是人之所以区别于物的特点。

> 一切事情是要人做的...做就必须先有人根据客观事实，引出思想、道理、意见，提出计划、方针、政策、战略、战术，方能做得好。

> 战争指挥员活动的舞台，必须建筑在客观条件的许可之上，然而他们凭借这个舞台，却可以导演出很多有声有色、威武雄壮的戏剧来。

> 坐着不动，只有被灭亡，没有持久战，也没有最后胜利。

> 指导战争的人们不能超越客观条件许可的限度期求战争的胜利，然而可以而且必须在客观条件的限度之内，能动地争取战争的胜利。
```

```markdown
# 领导方法法

> 来源：《关于领导方法的若干问题》（1943年6月）

## 核心原则

领导方法法是毛泽东对领导工作的方法论总结。核心命题：**一般与个别相结合，领导与群众相结合**

### 核心观点

1. **两个结合**：一般号召与个别指导相结合，领导骨干与广大群众相结合
2. **从群众中来，到群众中去**：集中群众意见，形成决策，再回到群众中坚持下去
3. **突破一点，推动全局**：取得经验后推广，实现全面突破
4. **一个中心工作**：在一定时间内只能有一个中心工作，统筹全局

### 适用场景

- 团队管理与项目推进
- 变革管理与落地执行
- 优先级管理与资源分配
- 经验总结与推广复制

---

## 分析框架

### 两个结合的领导方法

```
┌─────────────────────────────────────────────────────────────────┐
│                    一般与个别相结合                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    一般号召                                                     │
│    ├── 提出任务和目标                                           │
│    ├── 明确方向和要求                                           │
│    └── 动员和教育群众                                           │
│           │                                                     │
│           ▼                                                     │
│    个别指导                                                     │
│    ├── 深入具体单位                                             │
│    ├── 解决实际问题                                             │
│    └── 取得经验后推广                                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    领导与群众相结合                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    领导骨干                                                     │
│    ├── 团结积极分子组成核心                                     │
│    ├── 保持核心的动态调整                                       │
│    └── 防止脱离群众                                             │
│           │                                                     │
│           ▼                                                     │
│    广大群众                                                     │
│    ├── 从群众中集中意见                                         │
│    ├── 到群众中坚持下去                                         │
│    └── 形成无限循环                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：从群众中集中意见

- 深入了解群众的实际需求
- 收集群众的意见和建议
- 分析整合，形成初步方案

**关键问题清单**：
- [ ] 我是否真正听取了群众的意见？
- [ ] 我是否找到了积极分子组成领导核心？
- [ ] 我提出的任务是否符合实际？

### 步骤二：形成决策和计划

- 将群众意见系统化
- 明确中心工作和次要工作
- 制定具体的行动计划

### 步骤三：个别指导取得经验

- 选择典型单位试点
- 深入一线解决实际问题
- 总结成功经验

### 步骤四：推广经验推动全局

- 将个别经验上升为一般方法
- 在更大范围内推广
- 形成新的循环

---

## 应用示例

### 示例 1：项目推进

**场景**：推动新的工作流程在团队落地。

**一般号召**：向全团队说明新流程的意义和要求

**个别指导**：选择一个小组试点，深入指导

**取得经验**：发现实际问题，总结解决方案

**推动全局**：将试点经验推广到其他小组

### 示例 2：优先级管理

**场景**：团队同时面临多个任务，资源分散。

**错误做法**：同时推进多个"中心工作"，结果都不理想

**正确做法**：
- 明确当前阶段的中心工作
- 其他工作处于配合地位
- 集中资源突破一点
- 取得成果后再转向下一个中心

---

## 详细指导

### 应用技巧

1. **只做一个中心**：一定时间内只有一个中心工作，避免多中心
2. **动态调整骨干**：在斗争中不断提拔积极分子，替换腐化分子
3. **一般号召后紧跟个别指导**：不能只停留在嘴上、纸上、会议上
4. **形成无限循环**：一次比一次更正确、更生动、更丰富

### 常见误区

| 误区 | 表现 | 纠正方法 |
|------|------|----------|
| 官僚主义（一般无个别） | 只做号召不深入指导 | 紧接着从事个别的具体指导 |
| 官僚主义（领导无群众） | 只有骨干无群众结合 | 团结积极分子，密切联系群众 |
| 多中心工作 | 同时搞多个中心 | 一定时间只一个中心工作 |
| 骨干固化 | 领导班子一成不变 | 不断提拔积极分子，替换腐化分子 |

### 与其他方法论的配合

- **群众路线法**：领导方法法的核心就是群众路线的具体化
- **调查研究法**：了解群众意见的方法
- **聚焦突破法**：一个中心工作的具体应用

---

## 经典语录

> 我们共产党人无论进行何项工作，有两个方法是必须采用的，一是一般和个别相结合，二是领导和群众相结合。

> 在我党的一切实际工作中，凡属正确的领导，必须是从群众中来，到群众中去。

> 在任何一个地区内，不能同时有许多中心工作，在一定时间内只能有一个中心工作，辅以别的第二位、第三位的工作。

> 突破一点，取得经验，然后利用这种经验去指导其它单位，就可以说已经解决了工作中的大部分问题。

> 如此无限循环，一次比一次地更正确、更生动、更丰富。这就是马克思主义的认识论。
```
