# 调查研究法

> 来源：《反对本本主义》（1930年5月）

## 核心原则

调查研究法是实事求是的具体实践方法。核心命题：**"没有调查，没有发言权"**

### 核心观点

1. **调查的必要性**：不做调查，结论就是空想
2. **调查的态度**："眼睛向下"，深入一线
3. **调查的方法**：讨论式调查，多听多问
4. **调查的目的**：解决问题，不是为了调查而调查

### 适用场景

- 决策前需要了解真实情况
- 问题诊断需要找到根本原因
- 方案设计需要用户反馈
- 评估效果需要收集数据

## 分析框架

### 调查研究七法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 1. 讨论式调查 | 开调查会，集体讨论 | 获取多方视角 |
| 2. 邀请知情者 | 请了解情况的人参加 | 获取真实信息 |
| 3. 多与人交谈 | 广泛接触不同人群 | 全面了解情况 |
| 4. 定调查纲目 | 明确调查目的和问题 | 提高调查效率 |
| 5. 亲自出马 | 不依赖汇报，亲自调研 | 获取第一手资料 |
| 6. 深入一个点 | 选择典型深入调查 | "解剖麻雀" |
| 7. 自己记录 | 亲自记录，不假手他人 | 确保信息准确 |

### 步骤一：明确调查目的

- 为什么要调查？
- 想解决什么问题？
- 需要哪些信息？

### 步骤二：确定调查对象

- 谁掌握关键信息？
- 谁能提供不同视角？
- 如何接触这些对象？

### 步骤三：设计调查内容

**调查纲目示例**：

```markdown
## 关于 [问题] 的调查纲目

### 一、基本情况
- 现状是什么？
- 历史演变如何？

### 二、问题分析
- 主要问题有哪些？
- 问题的原因是什么？
- 各方看法如何？

### 三、解决思路
- 有哪些可能的方案？
- 各方案的利弊？
- 推荐方案是什么？
```

### 步骤四：实施调查

- 开调查会，邀请相关人员
- 个别访谈，深入了解
- 实地观察，获取直观感受

### 步骤五：整理分析

- 汇总调查结果
- 分析问题规律
- 形成调查结论

## 应用示例

### 示例 1：用户需求调研

**场景**：产品需要了解用户的真实需求。

**错误做法**：
- 只看数据报表，不接触用户
- 只听销售反馈，不直接调研
- 用问卷代替深入访谈

**正确做法**：
- 设计调查纲目
- 选择典型用户深入访谈
- 亲自参与用户调研过程
- 记录用户原话和行为

**调查结果**：发现用户的真实需求与团队假设存在差距，调整产品方向。

### 示例 2：团队问题诊断

**场景**：团队效率低下，需要找到原因。

**调查方法**：
- 与不同层级人员分别座谈
- 了解一线人员的真实想法
- 分析历史数据

**调查发现**：问题根源不是能力不足，而是流程不合理。

## 详细指导

### 调查的态度

1. **眼睛向下**：放下架子，虚心学习
2. **甘当小学生**：承认自己不懂，虚心请教
3. **实事求是**：有好说好，有坏说坏
4. **全面客观**：不带着结论找证据

### 调查的技巧

| 技巧 | 说明 |
|------|------|
| 先听后问 | 让对方充分表达，不急于打断 |
| 问开放问题 | 不问是非题，问"为什么""怎么看" |
| 追问细节 | 对关键问题深入追问 |
| 记录原话 | 准确记录，不加入主观理解 |
| 交叉验证 | 从多个来源验证信息 |

### 常见误区

| 误区 | 表现 | 纠正方法 |
|------|------|----------|
| 走过场 | 形式主义，为调查而调查 | 明确目的，解决问题导向 |
| 带结论调查 | 先有结论，找证据支撑 | 放下预设，客观收集信息 |
| 只听汇报 | 不深入一线，只听层层汇报 | 亲自出马，直接接触 |
| 以偏概全 | 只调查一个点就下结论 | 多点调查，全面了解 |

### 调查与决策的关系

```
调查 → 情况了解 → 问题分析 → 方案制定 → 决策
```

没有调查就没有决策的资格。

### 与其他方法论的配合

- **实事求是法**：调查研究是实事求是的具体实践
- **阶级立场分析法**：通过调查了解各方立场
- **矛盾分析法**：通过调查识别主要矛盾

## 经典语录

> 没有调查，没有发言权。

> 你对于那个问题不能解决吗？那末，你就去调查那个问题的现状和它的历史吧！

> 一切结论产生于调查情况的末尾，而不是在它的先头。

> 调查就像"十月怀胎"，解决问题就像"一朝分娩"。调查就是解决问题。

> 我们一定要纠正脱离实际情况的本本主义，我们必须根据实际情况来决定我们的工作方针。

> 离开实际调查就要产生唯心的阶级估量和唯心的工作指导，那末，它的结果，不是机会主义，便是盲动主义。
