# 统筹兼顾法

> 来源：《论十大关系》（1956年4月25日）

## 核心原则

统筹兼顾法的核心是：**处理好各种关系，实现协调发展**

### 核心观点

1. **系统思维**：看到事物之间的相互联系
2. **平衡协调**：不片面强调一点，忽视其他
3. **重点论与两点论**：既抓重点，又统筹全局
4. **辩证统一**：在对立中求统一

### 适用场景

- 制定复杂系统方案
- 资源平衡与协调
- 处理多方利益关系
- 战略规划与决策

## 分析框架

### 十大关系（方法论启示）

| 关系 | 辩证处理 |
|------|----------|
| 重工业与轻工业、农业 | 在发展重工业的同时，注意发展轻工业和农业 |
| 沿海与内地 | 好好利用和发展沿海的工业基地，支持内地发展 |
| 经济建设与国防建设 | 在增强国防力量的基础上增加经济建设费用 |
| 国家、生产单位、生产者 | 兼顾三者的利益 |
| 中央与地方 | 在中央统一领导下，扩大地方的权力 |
| 汉族与少数民族 | 诚心诚意帮助少数民族发展 |
| 党与非党 | 长期共存，互相监督 |
| 革命与反革命 | 肯定成绩，纠正错误 |
| 是非 | 分清是非，团结同志 |
| 中国与外国 | 学习好的，拒绝坏的 |

### 步骤一：识别关键关系

- 这个问题涉及哪些方面？
- 各方面之间有什么关系？
- 有哪些对立统一的关系？

### 步骤二：分析矛盾关系

- 主要矛盾是什么？
- 矛盾双方如何统一？
- 如何在对立中求平衡？

### 步骤三：制定协调方案

- 如何统筹各方？
- 如何平衡利益？
- 如何实现共赢？

## 应用示例

### 示例 1：产品规划

**场景**：产品规划需要平衡多个目标。

**关键关系**：
- 功能丰富度 vs 开发成本
- 用户体验 vs 商业目标
- 短期收益 vs 长期价值

**统筹方案**：
- 在保证核心体验的前提下控制成本
- 寻找用户体验与商业目标的结合点
- 短期收益支撑长期投入

### 示例 2：团队管理

**场景**：团队管理需要平衡多方利益。

**关键关系**：
- 个人发展 vs 团队目标
- 效率 vs 质量
- 创新风险 vs 稳定运行

**统筹方案**：
- 让个人目标与团队目标一致
- 建立质量保障下的效率提升
- 控制风险前提下的创新尝试

## 详细指导

### 统筹兼顾的原则

1. **系统思考**：把问题放到更大的系统中看
2. **动态平衡**：平衡不是静态的，是动态调整的
3. **抓住关键**：统筹不是平均，要抓主要矛盾
4. **留有余地**：不把事情做绝，保持弹性

### 常见误区

| 误区 | 纠正方法 |
|------|----------|
| 单打一 | 看到各方面的联系 |
| 平均主义 | 抓重点，兼顾其他 |
| 静止平衡 | 动态调整 |
| 顾此失彼 | 系统思考 |
