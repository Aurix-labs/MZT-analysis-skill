# 领导方法法

> 来源：《关于领导方法的若干问题》（1943年6月）

## 核心原则

领导方法法是毛泽东对领导工作的方法论总结。核心命题：**一般与个别相结合，领导与群众相结合**

### 核心观点

1. **两个结合**：一般号召与个别指导相结合，领导骨干与广大群众相结合
2. **从群众中来，到群众中去**：集中群众意见，形成决策，再回到群众中坚持下去
3. **突破一点，推动全局**：取得经验后推广，实现全面突破
4. **一个中心工作**：在一定时间内只能有一个中心工作，统筹全局

### 适用场景

- 团队管理与项目推进
- 变革管理与落地执行
- 优先级管理与资源分配
- 经验总结与推广复制

## 分析框架

### 两个结合的领导方法

```
│                    一般与个别相结合                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│    ├── 提出任务和目标                                           │
│    ├── 明确方向和要求                                           │
│    └── 动员和教育群众                                           │
│           │                                                     │
│           ▼                                                     │
│    个别指导                                                     │
│    ├── 深入具体单位                                             │
│    ├── 解决实际问题                                             │
│    └── 取得经验后推广                                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    领导与群众相结合                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    领导骨干                                                     │
│    ├── 团结积极分子组成核心                                     │
│    ├── 保持核心的动态调整                                       │
│    └── 防止脱离群众                                             │
│           │                                                     │
│           ▼                                                     │
│    广大群众                                                     │
│    ├── 从群众中集中意见                                         │
│    ├── 到群众中坚持下去                                         │
│    └── 形成无限循环                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：从群众中集中意见

- 深入了解群众的实际需求
- 收集群众的意见和建议
- 分析整合，形成初步方案

**关键问题清单**：
- [ ] 我是否真正听取了群众的意见？
- [ ] 我是否找到了积极分子组成领导核心？
- [ ] 我提出的任务是否符合实际？

### 步骤二：形成决策和计划

- 将群众意见系统化
- 明确中心工作和次要工作
- 制定具体的行动计划

### 步骤三：个别指导取得经验

- 选择典型单位试点
- 深入一线解决实际问题
- 总结成功经验

### 步骤四：推广经验推动全局

- 将个别经验上升为一般方法
- 在更大范围内推广
- 形成新的循环

## 应用示例

### 示例 1：项目推进

**场景**：推动新的工作流程在团队落地。

**一般号召**：向全团队说明新流程的意义和要求

**个别指导**：选择一个小组试点，深入指导

**取得经验**：发现实际问题，总结解决方案

**推动全局**：将试点经验推广到其他小组

### 示例 2：优先级管理

**场景**：团队同时面临多个任务，资源分散。

**错误做法**：同时推进多个"中心工作"，结果都不理想

**正确做法**：
- 明确当前阶段的中心工作
- 其他工作处于配合地位
- 集中资源突破一点
- 取得成果后再转向下一个中心

## 详细指导

### 应用技巧

1. **只做一个中心**：一定时间内只有一个中心工作，避免多中心
2. **动态调整骨干**：在斗争中不断提拔积极分子，替换腐化分子
3. **一般号召后紧跟个别指导**：不能只停留在嘴上、纸上、会议上
4. **形成无限循环**：一次比一次更正确、更生动、更丰富

### 常见误区

| 误区 | 表现 | 纠正方法 |
|------|------|----------|
| 官僚主义（一般无个别） | 只做号召不深入指导 | 紧接着从事个别的具体指导 |
| 官僚主义（领导无群众） | 只有骨干无群众结合 | 团结积极分子，密切联系群众 |
| 多中心工作 | 同时搞多个中心 | 一定时间只一个中心工作 |
| 骨干固化 | 领导班子一成不变 | 不断提拔积极分子，替换腐化分子 |

### 与其他方法论的配合

- **群众路线法**：领导方法法的核心就是群众路线的具体化
- **调查研究法**：了解群众意见的方法
- **聚焦突破法**：一个中心工作的具体应用

## 经典语录

> 我们共产党人无论进行何项工作，有两个方法是必须采用的，一是一般和个别相结合，二是领导和群众相结合。

> 在我党的一切实际工作中，凡属正确的领导，必须是从群众中来，到群众中去。

> 在任何一个地区内，不能同时有许多中心工作，在一定时间内只能有一个中心工作，辅以别的第二位、第三位的工作。

> 突破一点，取得经验，然后利用这种经验去指导其它单位，就可以说已经解决了工作中的大部分问题。

> 如此无限循环，一次比一次地更正确、更生动、更丰富。这就是马克思主义的认识论。
