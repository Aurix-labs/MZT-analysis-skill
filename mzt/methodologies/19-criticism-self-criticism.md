# 批评与自我批评法

> 来源：《关于纠正党内的错误思想》（1929年12月）、《论联合政府》（1945年4月）

## 核心原则

批评与自我批评法是毛泽东思想中的**自我迭代机制**。核心命题：**通过自我革命实现持续进步**

### 核心观点

1. **团结—批评—团结**：从团结的愿望出发，经过批评，达到新的团结
2. **知无不言，言无不尽**：坦诚地指出问题
3. **有则改之，无则加勉**：虚心接受批评
4. **自我革命**：敢于否定自己，才能超越自己

### 适用场景

- 团队复盘与改进
- 个人成长与反思
- 组织诊断与变革
- 关系修复与沟通

## 分析框架

### 批评与自我批评的流程

```
┌─────────────────────────────────────────────────────────────────┐
│                    团结 — 批评 — 团结                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    团结（出发点）                                               │
│    ├── 从良好的愿望出发                                         │
│    ├── 目的是帮助改进，不是打击                                  │
│    └── 保持团结的基调                                           │
│           │                                                     │
│           ▼                                                     │
│    批评（过程）                                                 │
│    ├── 指出具体问题                                             │
│    ├── 分析问题原因                                             │
│    ├── 提出改进建议                                             │
│    └── 允许申辩和讨论                                           │
│           │                                                     │
│           ▼                                                     │
│    团结（新境界）                                               │
│    ├── 在更高水平上团结                                         │
│    ├── 问题得到解决                                             │
│    └── 关系更加牢固                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：确立出发点

- 批评的目的是什么？（帮助改进）
- 是否从团结的愿望出发？
- 批评的方式是否恰当？

### 步骤二：具体问题具体分析

- 问题是什么？（具体表现）
- 为什么会有这个问题？（根源分析）
- 如何改进？（解决方案）

### 步骤三：开展批评

**批评的原则**：
- 事实清楚，不凭空臆断
- 区分性质，不无限上纲
- 与人为善，不恶语伤人
- 有则改之，无则加勉

### 步骤四：接受批评

**接受批评的态度**：
- 虚心听取，不急于反驳
- 有则改之，无则加勉
- 即使批评不完全准确，也要反思

## 应用示例

### 示例 1：团队复盘

**场景**：项目结束后进行团队复盘。

**批评与自我批评应用**：
- 每个人先自我批评：我做错了什么？
- 然后互相批评：我看到的问题是什么？
- 讨论如何改进
- 形成新的共识

### 示例 2：个人反思

**场景**：定期进行个人成长反思。

**自我批评框架**：
- 这段时间我有哪些做得不好的地方？
- 原因是什么？
- 如何改进？
- 下次遇到类似情况怎么办？

## 详细指导

### 批评的艺术

| 原则 | 说明 |
|------|------|
| 惩前毖后 | 批评是为了以后不犯错 |
| 治病救人 | 批评的目的是帮助，不是打击 |
| 与人为善 | 批评要出于善意 |
| 实事求是 | 批评要基于事实 |

### 自我批评的境界

1. **敢于承认错误**：不遮掩、不推诿
2. **深刻分析原因**：找到思想根源
3. **拿出改进措施**：有具体的行动计划
4. **接受他人监督**：让改进过程透明

### 常见误区

| 误区 | 纠正方法 |
|------|----------|
| 批评变成攻击 | 对事不对人，目的是改进 |
| 自我批评走过场 | 深挖根源，触及灵魂 |
| 只批评不自我批评 | 以身作则，先从自己开始 |
| 怕得罪人不敢批评 | 从团结的愿望出发，敢于直言 |

## 经典语录

> 有无认真的自我批评，也是我们和其他政党互相区别的显著的标志之一。

> 房子是应该经常打扫的，不打扫就会积满了灰尘；脸是应该经常洗的，不洗也就会灰尘满面。

> 我们同志的思想，我们党的工作，也会沾染灰尘的，也应该打扫和洗涤。

> "流水不腐，户枢不蠹"，是说它们在不停的运动中抵抗了微生物或其他生物的侵蚀。

> 以"惩前毖后，治病救人"为宗旨的整风运动之所以发生了很大的效力，就是因为我们在这个运动中展开了正确的而不是歪曲的、认真的而不是敷衍的批评和自我批评。
