# 宗旨导向法

> 来源：《为人民服务》（1944年9月8日）

## 核心原则

宗旨导向法的核心是：**明确根本宗旨，以此指引一切行动**

### 核心观点

1. **根本宗旨**：为人民服务是根本出发点
2. **价值导向**：一切从人民利益出发
3. **意义建构**：找到工作的意义和价值
4. **牺牲精神**：为宗旨可以牺牲个人利益

### 适用场景

- 确立使命愿景
- 价值体系建设
- 团队凝聚力打造
- 工作意义寻找

## 分析框架

### 为人民服务的核心内涵

```
┌─────────────────────────────────────────────────────────────────┐
│                    为什么做                                      │
│    为人民服务 — 根本宗旨                                        │
├─────────────────────────────────────────────────────────────────┤
│                    做什么                                        │
│    有利于人民的事 — 行动标准                                    │
├─────────────────────────────────────────────────────────────────┤
│                    怎么做                                        │
│    全心全意 — 态度要求                                          │
├─────────────────────────────────────────────────────────────────┤
│                    结果是什么                                    │
│    人民满意 — 检验标准                                          │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：明确宗旨

- 我们存在的意义是什么？
- 我们为谁服务？
- 我们的核心价值是什么？

### 步骤二：对齐行动

- 当前的工作是否符合宗旨？
- 有没有偏离宗旨？
- 如何更好地实现宗旨？

### 步骤三：检验效果

- 服务对象是否满意？
- 是否真正创造了价值？
- 是否有需要改进的地方？

---

## 应用示例

### 示例 1：产品团队

**场景**：产品团队需要明确工作方向。

**宗旨导向应用**：
- **宗旨**：为用户创造价值
- **对齐行动**：每个功能都要问"对用户有什么价值？"
- **检验效果**：用户是否满意？是否解决了用户问题？

### 示例 2：个人职业

**场景**：个人思考职业发展方向。

**宗旨导向应用**：
- **宗旨**：我想为这个世界贡献什么？
- **对齐行动**：当前工作是否符合我的宗旨？
- **检验效果**：我是否在向目标靠近？

## 详细指导

### 宗旨的力量

1. **方向指引**：在迷茫时提供方向
2. **动力源泉**：在困难时提供动力
3. **决策依据**：在选择时提供标准
4. **意义感**：赋予工作以意义

### 全心全意的态度

- 不只是"做"，而是"做好"
- 不只是"完成任务"，而是"创造价值"
- 不只是"尽力而为"，而是"全力以赴"

### 宗旨与实践的关系

| 层次 | 内容 |
|------|------|
| 理念层 | 明确宗旨 |
| 制度层 | 建立符合宗旨的机制 |
| 行为层 | 每个行动都体现宗旨 |
| 效果层 | 用成果检验宗旨实现 |

### 与其他方法论的配合

- **群众路线法**：宗旨的具体实践方法
- **实事求是法**：从实际出发实现宗旨
- **持之以恒法**：为宗旨持续努力
