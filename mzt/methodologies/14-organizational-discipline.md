# 组织纪律法

> 来源：《反对自由主义》（1937年9月7日）

## 核心原则

组织纪律法的核心是：**原则性高于人情关系，组织利益高于个人利益**

### 核心观点

1. **反对自由主义**：自由主义削弱组织战斗力
2. **原则性**：在大是大非问题上坚持原则
3. **组织纪律**：个人服从组织，少数服从多数
4. **批评与自我批评**：通过思想斗争解决分歧

### 适用场景

- 团队建设与组织管理
- 文化建设与价值观统一
- 冲突处理与纪律执行
- 个人修养与自我反思

## 分析框架

### 自由主义的十一种表现

| 类型 | 表现 | 危害 |
|------|------|------|
| 1 | 为了保持一团和气，不开展批评 | 是非不分，问题积累 |
| 2 | 不负责任背后批评，当面不说 | 破坏团结，制造隔阂 |
| 3 | 事不关己，高高挂起 | 缺乏责任感，问题扩大 |
| 4 | 不服从命令，个人意见第一 | 破坏纪律，影响执行 |
| 5 | 搞小团体，不团结 | 分裂组织，削弱力量 |
| 6 | 听到错误言论不报告 | 纵容错误，危害组织 |
| 7 | 见群众不宣传，不关心 | 脱离群众，失去支持 |
| 8 | 见损害群众利益不斗争 | 损害形象，失去信任 |
| 9 | 办事不认真，敷衍应付 | 效率低下，质量不高 |
| 10 | 自以为对，不接受意见 | 固步自封，无法进步 |
| 11 | 自己错了也不改正 | 错误延续，后果严重 |

### 步骤一：识别自由主义表现

- 自己是否存在这些问题？
- 团队中是否存在这些问题？
- 这些问题的危害是什么？

### 步骤二：分析根源

- 为什么会出现这些问题？
- 是认识问题还是立场问题？
- 有什么深层次原因？

### 步骤三：纠正改进

- 提高认识，认清危害
- 坚持原则，敢于斗争
- 接受批评，自我改正

## 应用示例

### 示例 1：团队协作问题

**场景**：团队存在"老好人"现象，问题没人指出。

**自由主义表现**：为了保持一团和气，不开展批评

**危害**：
- 问题被掩盖
- 团队无法进步
- 错误重复发生

**纠正方法**：
- 建立"对事不对人"的文化
- 鼓励建设性批评
- 领导以身作则

### 示例 2：执行纪律问题

**场景**：团队成员不遵守规定，"各自为政"。

**自由主义表现**：不服从命令，个人意见第一

**纠正方法**：
- 明确规则和底线
- 对违规行为严肃处理
- 建立公平的奖惩机制

## 详细指导

### 克服自由主义的方法

1. **提高认识**：认清自由主义的危害
2. **坚持原则**：在大是大非问题上不让步
3. **积极斗争**：对错误行为敢于批评
4. **自我批评**：勇于承认和改正自己的错误

### 组织纪律的原则

| 原则 | 说明 |
|------|------|
| 个人服从组织 | 个人利益服从组织利益 |
| 少数服从多数 | 尊重集体决策 |
| 下级服从上级 | 保持执行效率 |
| 全党服从中央 | 保持统一 |

### 与"一团和气"的区别

- **真正的团结**：建立在原则基础上，通过批评与自我批评实现
- **一团和气**：无原则的调和，掩盖矛盾，不是真正的团结

> 自由主义取消思想斗争，主张无原则的和平，结果是腐朽庸俗的作风发生。

> 我们主张积极的思想斗争，因为它是达到党内和革命团体内的团结使之利于战斗的武器。

> 一个共产党员，应该是襟怀坦白，忠实，积极，以革命利益为第一生命，以个人利益服从革命利益。

> 共产党员应该随时准备坚持真理，因为任何真理都是符合于人民利益的；共产党员应该随时准备修正错误，因为任何错误都是不符合于人民利益的。
