# 聚焦突破法

> 来源：《集中优势兵力，各个歼灭敌人》（1946年9月16日）

## 核心原则

聚焦突破法的核心是：**集中资源于关键点，实现重点突破**

### 核心观点

1. **集中原则**：不搞平均分配，集中资源于关键点
2. **各个击破**：一次解决一个问题，不贪多
3. **以多胜少**：在局部形成优势，实现突破
4. **积小胜为大胜**：通过连续的小突破实现大目标

### 适用场景

- 资源有限，需要高效分配
- 目标众多，需要确定优先级
- 竞争激烈，需要建立局部优势
- 执行困难，需要重点突破

---

## 分析框架

### 集中优势兵力的原则

```
┌─────────────────────────────────────────────────────────────────┐
│                    全局劣势 → 局部优势                          │
├─────────────────────────────────────────────────────────────────┤
│ • 总体资源不如对手                                              │
│ • 但可以在关键点集中更多资源                                    │
│ • 在局部形成以多打少的优势                                      │
│ • 实现突破后再转向下一个目标                                    │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：识别关键点

- 哪个问题最关键？
- 哪个市场最有潜力？
- 哪个能力最核心？

### 步骤二：评估资源

- 我有多少资源？
- 对手投入多少？
- 需要多少才能突破？

### 步骤三：集中资源

- 减少非核心投入
- 调配资源到关键点
- 确保在关键点形成优势

### 步骤四：各个击破

- 一次只攻打一个目标
- 攻下后迅速转向下一个
- 积累胜利，扩大战果

---

## 应用示例

### 示例 1：产品开发优先级

**场景**：团队资源有限，需求很多。

**错误做法**：平均分配资源，每个需求都做一点

**聚焦突破**：
- 识别核心需求（用户价值最高）
- 集中资源攻克核心需求
- 做好一个再做下一个

### 示例 2：市场拓展

**场景**：新进入一个市场，资源有限。

**聚焦突破**：
- 选择一个细分市场重点突破
- 在这个细分市场建立领先地位
- 然后向相关市场扩展

## 详细指导

### 集中的度

| 情况 | 集中程度 |
|------|----------|
| 突破关键问题 | 高度集中 |
| 维持运营 | 适度分散 |
| 探索机会 | 小规模试验 |

### 各个击破的节奏

1. **选准目标**：确保值得投入
2. **集中力量**：确保能够突破
3. **快速突破**：避免僵持消耗
4. **迅速转移**：不恋战，扩大战果

### 常见误区

| 误区 | 纠正方法 |
|------|----------|
| 贪多求全 | 砍掉次要目标，聚焦核心 |
| 平均用力 | 识别关键点，集中投入 |
| 目标过大 | 分解为可突破的小目标 |
| 战线过长 | 收缩战线，逐个击破 |

## 经典语录

> 集中优势兵力、各个歼灭敌人的作战方法，不但必须应用于战役的部署方面，而且必须应用于战术的部署方面。

> 在战术的部署方面，当着敌人使用许多个旅（或团）分几路向我军前进的时候，我军必须集中绝对优势的兵力，即集中六倍、或五倍、或四倍于敌的兵力，至少也要有三倍于敌的兵力。

> 集中兵力看来容易，实行颇难。人人皆知以多胜少是最好的办法，然而很多人不能做，相反地每每分散兵力，原因就在于指导者缺乏战略头脑，不为复杂的环境所迷惑。

> 这种战法的效果是：一能全歼；二能速决。
