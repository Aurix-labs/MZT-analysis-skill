# 统一战线法

> 来源：《目前抗日统一战线中的策略问题》（1940年3月）、《论政策》（1940年12月）

## 核心原则

统一战线法是毛泽东思想的"三大法宝"之一。核心命题：**团结一切可以团结的力量**

### 核心观点

1. **团结多数，孤立少数**：把朋友搞得多多的，把敌人搞得少少的
2. **利用矛盾，争取多数**：在敌人内部寻找可以利用的矛盾
3. **有理、有利、有节**：斗争要有道理、有利益、有节制
4. **既联合又斗争**：统一战线不是无原则的妥协

### 适用场景

- 需要争取各方支持
- 面对强大对手，需要建立联盟
- 利益博弈，需要分化对手
- 推动变革，需要凝聚共识

## 分析框架

### 统一战线的核心策略

```
┌─────────────────────────────────────────────────────────────────┐
│                    统一战线的三个层次                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    第一层：核心力量                                             │
│    ├── 必须依靠的基本力量                                       │
│    └── 利益完全一致的同盟者                                     │
│                                                                 │
│    第二层：争取力量                                             │
│    ├── 可以争取的中间力量                                       │
│    └── 通过利益交换获得支持                                     │
│                                                                 │
│    第三层：分化力量                                             │
│    ├── 可以暂时合作的对象                                       │
│    └── 利用其矛盾，减少阻力                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：分析力量格局

- 谁是核心支持者？
- 谁是中间力量？
- 谁是反对者？
- 反对者内部是否有矛盾？

### 步骤二：确定统战对象

| 力量类型 | 立场 | 策略 |
|----------|------|------|
| 核心力量 | 坚定支持 | 依靠 |
| 中间力量 | 态度摇摆 | 争取 |
| 可分化力量 | 有条件合作 | 利用 |
| 坚定反对 | 顽固对立 | 孤立 |

### 步骤三：制定统战策略

- 核心力量：加强团结，巩固联盟
- 中间力量：利益绑定，争取转化
- 可分化力量：利用矛盾，暂时合作
- 坚定反对：坚决斗争，孤立打击

### 步骤四：把握斗争原则

- **有理**：站在道义制高点
- **有利**：斗争要获得实际利益
- **有节**：适可而止，留有余地

## 应用示例

### 示例 1：推动组织变革

**场景**：在公司推行一项改革，面临阻力。

**统一战线分析**：
- 核心力量：改革的直接受益者
- 中间力量：态度摇摆的中层管理者
- 可分化力量：有个人诉求的高管
- 坚定反对：既得利益受损者

**策略**：
- 团结受益者作为推动力量
- 通过利益交换争取中间派
- 利用高管之间的分歧
- 对顽固反对者进行孤立

### 示例 2：市场竞争

**场景**：面对行业巨头的竞争压力。

**统一战线分析**：
- 谁是潜在的盟友？
- 竞争对手之间有什么矛盾？
- 如何构建竞合关系？

**策略**：联合中小玩家，利用巨头之间的矛盾，在细分领域建立优势。

## 详细指导

### 统一战线的三个原则

| 原则 | 说明 | 应用 |
|------|------|------|
| 有理 | 占据道义制高点 | 让对方在舆论上处于劣势 |
| 有利 | 斗争要获得实际利益 | 不打无把握之仗 |
| 有节 | 适可而止，留有余地 | 不把对方逼到绝境 |

### 统一战线与独立自主的关系

- **独立自主是前提**：没有独立自主，就会变成别人的附庸
- **统一战线是手段**：为了更好地实现独立自主的目标
- **既联合又斗争**：在联合中保持原则，在斗争中保持合作

### 常见误区

| 误区 | 纠正方法 |
|------|----------|
| 无原则妥协 | 坚持核心利益，灵活处理次要问题 |
| 只团结不斗争 | 斗争是团结的手段，团结是斗争的目的 |
| 关门主义 | 广泛团结，不搞"清一色" |

> 在日本帝国主义继续进攻，全国民族革命潮流继续高涨的情况下，国民党这一政策的转变，对于我们是有利的。

> 统一战线的原则有两个：第一个是团结，第二个是批评、教育和改造。

> 在统一战线中，斗争是团结的手段，团结是斗争的目的。以斗争求团结则团结存，以退让求团结则团结亡。

> 我们的同盟者有三种：一种是比较可靠的，一种是一时合作的，一种是不得不暂时合作的。
