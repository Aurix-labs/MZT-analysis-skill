# 独立自主法

> 来源：《井冈山的斗争》（1928年11月）、《论持久战》（1938年5月）

## 核心原则

独立自主法是毛泽东思想的灵魂之一。核心命题：**把命运掌握在自己手中**

### 核心观点

1. **自力更生为主，争取外援为辅**：立足自身力量，不依赖外部
2. **战略支点建设**：建立自己的"根据地"，有了立足之地才能发展
3. **保持主体性**：在合作中保持独立性，在依赖中保持主动权
4. **掌握核心能力**：关键时刻不被"卡脖子"

### 适用场景

- 制定发展战略，需要建立护城河
- 面对外部依赖，需要评估风险
- 资源有限，需要确定核心能力
- 合作谈判，需要保持独立地位

## 分析框架

### 独立自主的核心要素

```
┌─────────────────────────────────────────────────────────────────┐
│                    独立自主的四个维度                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    能力独立                      资源独立                       │
│    ├── 核心技能自己掌握          ├── 关键资源自己控制            │
│    ├── 不被技术封锁              ├── 不被供应链卡脖子            │
│    └── 可以持续迭代              └── 有替代方案                  │
│                                                                 │
│    决策独立                      发展独立                       │
│    ├── 不被外部意志左右          ├── 发展方向自己决定            │
│    ├── 保持战略定力              ├── 不被环境牵着走              │
│    └── 有说"不"的能力            └── 有长远规划                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 步骤一：识别依赖

- 哪些能力依赖外部？
- 哪些资源依赖外部？
- 哪些决策受制于人？
- 发展路径是否被外部左右？

### 步骤二：评估风险

| 依赖项 | 可替代性 | 断供影响 | 风险等级 |
|--------|----------|----------|----------|
| 依赖 A | 高/中/低 | 严重/一般/轻微 | 高/中/低 |
| 依赖 B | 高/中/低 | 严重/一般/轻微 | 高/中/低 |

### 步骤三：建立支点

- 什么是我的"井冈山"？（立足之地）
- 什么是我的"核心能力"？（不可替代）
- 什么是我的"护城河"？（竞争优势）

### 步骤四：制定策略

- 哪些必须自己掌握？
- 哪些可以合作但保持独立？
- 哪些可以完全依赖外部？

## 应用示例

### 示例 1：技术架构决策

**场景**：公司选择技术栈，是否使用开源项目？

**独立自主分析**：
- 核心技术是否掌握？
- 开源项目是否会持续维护？
- 如果项目停止维护，能否自己接手？

**结论**：核心能力自研，非核心可以使用开源，但要有替代方案。

### 示例 2：创业公司战略

**场景**：创业公司依赖大平台流量。

**独立自主分析**：
- 流量来源是否单一？
- 平台政策变化会影响多大？
- 是否有能力建立自己的用户池？

**结论**：利用平台流量起步，但要建设自有渠道，降低平台依赖。

## 详细指导

### 独立自主与对外开放的关系

| 关系 | 说明 |
|------|------|
| 不是闭关自守 | 独立自主不等于闭门造车 |
| 不是拒绝合作 | 合作中保持独立性 |
| 不是自给自足 | 关键环节自主，非关键可以外协 |

### "根据地"建设原则

1. **选择有利位置**：在自己的优势领域建立根据地
2. **深耕细作**：把根据地做深做透
3. **自我造血**：根据地要能持续发展
4. **可进可退**：保持战略弹性

### 与其他方法论的配合

- **统一战线法**：独立自主是统一战线的前提
- **群众路线法**：依靠自己的人民群众
- **聚焦突破法**：在核心领域建立优势
